Welcome to Wilber Claudio's Final project

My project uses both the Google Calendar API and the Google Apps Script API. The project will allow the user to annouce some event, which will be added to their google calendar by the app, and also inform a list of people who they would like to email about the event.(In order to use this Project the user must have a gmail account). Next Page

User Inputs:

  1. The event Name
  2. A description about the event
  3. The date of the event
  4. The start time of the event (in eastern time)
  5. The end time of the event (in eastern time)
  6. A list of emails (each email must be separated by a comma)

Email content:

  • Subject: `${EventName} on ${eventDate}`
  • Body: `${eventDesc}`

Endpoints used:

  1. https://accounts.google.com/o/oauth2/auth
  2. https://oauth2.googleapis.com/token
  3. https://www.googleapis.com/calendar/v3/calendars/primary/events
  4. https://script.googleapis.com/v1/scripts/${script_id}:run

Scopes used:

  1. https://www.googleapis.com/auth/script.send_mail
  2. https://www.googleapis.com/auth/calendar.events
